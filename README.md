# 🎨 Frontend UI & Animation Projects Collection

A collection of **modern UI components, animations, landing pages, and JavaScript mini projects** built using **HTML, CSS, and Vanilla JavaScript**.

This repository contains multiple frontend practice projects including **3D effects, sliders, loaders, navigation menus, interactive UI components, and small web applications**.

It is mainly created for:

* Practicing **Frontend Development**
* Learning **CSS Animations**
* Improving **JavaScript DOM manipulation**
* Building reusable **UI components**
* Creating **portfolio-ready mini projects**


### 2️⃣ Open the Project Folder

